# 1990年代：網際網路興起

## 時代背景
1990年代是網際網路快速發展的時期，從學術研究網路發展成為商業和個人使用的全球網路，改變了資訊傳播和商業模式。

## 主要科技趨勢
- **網際網路普及**：World Wide Web的發明和普及
- **瀏覽器戰爭**：Netscape vs Internet Explorer
- **電子商務興起**：Amazon等電商平台開始出現
- **搜尋引擎發展**：Google等搜尋引擎開始出現
- **個人電腦標準化**：Windows成為個人電腦標準

## 各公司發展狀況

### Apple（成立於1976年）
**發展狀況**：
- 1991年：與IBM合作開發PowerPC處理器
- 1996年：Apple收購NeXT，賈伯斯回歸
- 1997年：賈伯斯成為臨時CEO
- 1998年：推出iMac，重新設計個人電腦

**重要成就**：
- 賈伯斯回歸，開始公司改革
- 推出iMac，重新定義個人電腦設計
- 簡化產品線，專注核心產品

**挑戰**：
- 市佔率持續下降
- 財務困難
- 需要重新定位

### Microsoft（成立於1975年）
**發展狀況**：
- 1990年：推出Windows 3.0
- 1995年：推出Windows 95，改變個人電腦
- 1995年：推出Internet Explorer
- 1998年：推出Windows 98

**重要成就**：
- Windows成為個人電腦標準
- 開始進軍網際網路
- 建立強大的軟體生態系統

**挑戰**：
- 面臨反壟斷訴訟
- 瀏覽器戰爭的競爭
- 網際網路轉型的壓力

### NVIDIA（成立於1993年）
**發展狀況**：
- 1993年：在加州創立NVIDIA
- 1995年：推出NV1顯示卡
- 1997年：推出RIVA 128，開始成功
- 1999年：推出GeForce 256，革命性GPU

**重要成就**：
- 專注於3D圖形技術
- 發明GPU概念
- 開始在遊戲市場建立地位

**特色**：
- 技術創新驅動
- 遊戲市場定位
- 硬體技術專長

### Amazon（成立於1994年）
**發展狀況**：
- 1994年：在西雅圖車庫創立Amazon
- 1995年：網站正式上線，開始賣書
- 1997年：Amazon上市
- 1999年：開始賣更多商品類別

**重要成就**：
- 建立電商平台
- 建立客戶評價系統
- 開始建立物流網路

**特色**：
- 從賣書開始，逐步擴展
- 重視客戶體驗
- 建立信任機制

### 其他公司狀況

- **Meta（Facebook）**：尚未成立

## 競爭關係分析

### Microsoft vs Netscape
**競爭焦點**：
- **瀏覽器市場**：Internet Explorer vs Netscape Navigator
- **網際網路標準**：控制網際網路入口
- **平台整合**：瀏覽器與作業系統整合

**競爭結果**：
- Microsoft利用Windows優勢打敗Netscape
- 面臨反壟斷訴訟
- 瀏覽器戰爭影響後來的網際網路發展

### Apple vs Microsoft
**競爭焦點**：
- **個人電腦市場**：Mac vs PC
- **作業系統**：Mac OS vs Windows
- **設計理念**：封閉 vs 開放

**競爭結果**：
- Microsoft在個人電腦市場佔優勢
- Apple開始重新定位
- 賈伯斯回歸後開始改革

### 新興競爭者
- **Netscape**：瀏覽器市場競爭者
- **Yahoo**：入口網站競爭者
- **eBay**：電商平台競爭者
