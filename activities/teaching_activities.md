# 教學活動與互動內容設計

## 學習目標
- 讓中年級小學生了解美股七巨頭的發展歷程
- 理解科技產業的競爭與合作關係
- 培養創新思維和創業精神
- 認識科技對生活的影響

## 教學活動設計

### 第一單元：認識七巨頭

#### 活動1：創辦人配對遊戲
**目標**：讓學生認識各公司的創辦人
**材料**：創辦人照片、公司標誌、創辦人故事卡片
**活動流程**：
1. 展示七位創辦人的照片
2. 學生配對創辦人和公司
3. 分享創辦人的小故事
4. 討論創辦人的共同特質

**討論問題**：
- 這些創辦人有什麼共同點？
- 他們為什麼能成功？
- 你覺得哪個創辦人的故事最有趣？

#### 活動2：時間軸遊戲
**目標**：了解各公司的成立時間和發展順序
**材料**：公司成立時間卡片、大型時間軸海報
**活動流程**：
1. 學生將公司按成立時間排序
2. 在時間軸上標示重要事件
3. 討論不同時期的科技發展
4. 觀察公司發展的時代背景

**討論問題**：
- 為什麼這些公司都在特定時期成立？
- 科技發展如何影響公司成立？
- 不同時代的創業環境有什麼不同？

### 第二單元：公司特色與產品

#### 活動3：產品配對遊戲
**目標**：認識各公司的主要產品和服務
**材料**：產品圖片、公司標誌、產品描述
**活動流程**：
1. 展示各種產品圖片
2. 學生配對產品和公司
3. 討論產品的功能和特色
4. 分享使用這些產品的經驗

**討論問題**：
- 這些產品如何改變我們的生活？
- 你覺得哪個產品最有創意？
- 這些產品有什麼共同特色？

#### 活動4：公司文化角色扮演
**目標**：理解不同公司的文化特色
**材料**：公司文化描述、角色扮演卡片
**活動流程**：
1. 學生選擇扮演不同公司的員工
2. 根據公司文化特色進行角色扮演
3. 模擬公司會議或產品開發討論
4. 分享不同公司文化的感受

**討論問題**：
- 不同公司的文化有什麼不同？
- 你喜歡哪種公司文化？
- 公司文化如何影響產品設計？

### 第三單元：競爭與合作

#### 活動5：競爭關係圖表
**目標**：理解公司間的競爭關係
**材料**：公司標誌、競爭關係圖表、便利貼
**活動流程**：
1. 繪製公司競爭關係圖表
2. 學生標示競爭領域
3. 討論競爭的優缺點
4. 分析競爭對消費者的影響

**討論問題**：
- 競爭如何促進創新？
- 競爭對消費者有什麼好處？
- 如何避免惡性競爭？

#### 活動6：合作案例研究
**目標**：了解公司間的合作關係
**材料**：合作案例資料、合作關係圖表
**活動流程**：
1. 研究公司間的合作案例
2. 分析合作的動機和效果
3. 討論合作的優缺點
4. 設計新的合作方案

**討論問題**：
- 為什麼公司要合作？
- 合作有什麼好處和風險？
- 你覺得哪些公司應該合作？

### 第四單元：世代發展

#### 活動7：科技發展時間軸
**目標**：了解科技發展的世代變化
**材料**：各世代科技發展圖表、重要事件卡片
**活動流程**：
1. 製作科技發展時間軸
2. 標示各世代的重要科技
3. 討論科技對生活的影響
4. 預測未來科技發展

**討論問題**：
- 每個世代的科技有什麼特色？
- 科技發展如何改變生活？
- 你期待什麼樣的未來科技？

#### 活動8：創業模擬遊戲
**目標**：體驗創業過程
**材料**：創業模擬遊戲道具、角色卡片
**活動流程**：
1. 學生分組成立虛擬公司
2. 選擇產品和目標市場
3. 制定發展策略
4. 面對競爭和挑戰

**討論問題**：
- 創業需要什麼條件？
- 如何應對競爭和挑戰？
- 成功的創業者需要什麼特質？

### 第五單元：創新與未來

#### 活動9：未來產品設計
**目標**：培養創新思維
**材料**：設計工具、創意素材、展示板
**活動流程**：
1. 學生設計未來科技產品
2. 考慮產品的功能和特色
3. 製作產品原型或模型
4. 向同學展示和說明

**討論問題**：
- 你的產品解決什麼問題？
- 如何讓產品更受歡迎？
- 創新對社會有什麼影響？

#### 活動10：科技影響討論
**目標**：思考科技對社會的影響
**材料**：科技影響案例、討論指南
**活動流程**：
1. 討論科技的正負面影響
2. 分析科技對不同群體的影響
3. 提出改善科技影響的建議
4. 制定負責任使用科技的準則

**討論問題**：
- 科技如何改變我們的生活？
- 如何平衡科技發展和社會責任？
- 我們應該如何負責任地使用科技？

## 評量方式

### 形成性評量
- **課堂參與**：觀察學生參與活動的積極性
- **小組合作**：評估學生在小組活動中的表現
- **創意表現**：評價學生的創意和創新思維
- **理解程度**：通過討論了解學生的理解程度

### 總結性評量
- **專題報告**：學生選擇一家公司進行深入研究
- **創意作品**：設計未來科技產品或服務
- **反思日誌**：記錄學習心得和思考
- **小組展示**：展示學習成果和創意想法

## 教學建議

### 教學策略
- **故事化教學**：用創辦人故事吸引學生興趣
- **遊戲化學習**：通過遊戲提高學習動機
- **合作學習**：培養團隊合作能力
- **探究式學習**：鼓勵學生主動探索

### 注意事項
- 適應學生的認知水平
- 鼓勵多元觀點和創意
- 重視學生的情感體驗
- 培養批判性思維

### 延伸活動
- 參觀科技展覽
- 邀請創業者分享經驗
- 參與科技創新競賽
- 建立科技創新社團
